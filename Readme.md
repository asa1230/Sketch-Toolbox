#sketch插件超级管理工具#

*一键安装，卸载您的sketch插件，从此再不需要手动安装
*丰富方便，按安装量排序，快速找到好用的上手插件
*专业优化和改良，界面，使用教程，性能我们都做优化

提示: 如果你使用过程中遇到任何崩溃而且始终无法正常工作,请清理掉一下路径件 ```~/Library/Application Support/Sketch Toolbox/``` 的文，然后重启.
